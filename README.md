# 🌦️ Weather AI Agent

An autonomous AI agent that fetches real-time weather using tool-calling.

## 🚀 Features
- Multi-city weather support
- Memory (remembers last location)
- Autonomous tool usage
- Retry logic for stability

## 🛠️ Tech Stack
- Python
- OpenAI / Mistral API
- Requests

## ▶️ How to Run

```bash
pip install -r requirements.txt
python weather_agent.py
